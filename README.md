# Data Science Competitions

Ranked Kaggle-style competition entries. Each folder is a self-contained notebook + write-up; competition
datasets are **not** included (redistribution is not allowed by competition rules).

| Competition | Task | Result | Highlight |
|---|---|---|---|
| [ARA ITS 2025](./ara-its-2025) | Pothole semantic segmentation (CV) | **53 / 260** | SegFormer + custom boundary-weighted loss + low-confidence oversampling |
| [Dataquest 2025](./dataquest-2025) | Legal sentence-length regression (NLP) | **57 / 109** | OCR-robust Indonesian legal text + IndoBERT/TF-IDF dual embeddings |
| [Datavidia 2025](./datavidia-2025) | Air-quality index classification | **93 / 205** | Spatio-temporal grid reconstruction + weather/NDVI features |
